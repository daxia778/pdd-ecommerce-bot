# 店铺服务套餐与报价体系

## 店铺简介
本店专注于企业级PPT定制设计服务，提供从基础美化到高端发布会级别的全方位PPT制作。所有订单一对一沟通，由专业设计师团队进行定制化设计，拒绝模板套用。

---

## 服务套餐分级

### 🥉 基础美化套餐（入门级）
- **适用场景**：内部汇报、部门工作总结、培训课件、学生作业
- **服务内容**：排版优化 + 配色调整 + 基础图标配置 + 图片美化
- **价格区间**：**150元/页起**（最低5页起订）
- **最低起价**：约 **750元**（5页基础套餐）
- **交付周期**：3-5个工作日
- **修改次数**：免费修改2次，超出每次50元/页
- **客户须提供**：完整文字内容 + 参考素材（可选）

### 🥈 标准定制套餐（主力款）
- **适用场景**：企业宣传、年度汇报、奖项申报、项目提案、融资路演
- **服务内容**：主题设计 + 逻辑梳理 + 专业图表制作 + 定制化图标 + 高质量版权图片
- **价格区间**：**350元/页起**（最低8页起订）
- **最低起价**：约 **2800元**（8页标准套餐）
- **交付周期**：5-7个工作日
- **修改次数**：免费修改3次，超出每次100元/页
- **客户须提供**：基础思路 + 核心内容（可提供文案优化服务，另计）

### 🥇 高端定制套餐（旗舰款）
- **适用场景**：新品发布会、上市路演、品牌宣传、高端商务提案、政府汇报
- **服务内容**：全定制主题设计 + 内容提炼与优化 + 复杂动画效果 + 高端版权素材 + 彩排指导
- **价格区间**：**600-1000元/页起**（最低10页起订）
- **最低起价**：约 **6000元**（10页高端套餐）
- **交付周期**：7-14个工作日
- **修改次数**：免费修改5次，超出每次200元/页
- **客户须提供**：需求说明 + 品牌/VI规范（如有）

### 🎯 纯模板套餐（性价比款）
- **适用场景**：个人使用、简单活动、紧急场合
- **服务内容**：选用精品模板 + 内容填充 + 基础调整
- **价格区间**：**39-99元/页**，动态页面 **99元/页起**
- **最低起价**：约 **299元**（精品模板套餐整套）
- **交付周期**：1-2个工作日
- **修改次数**：免费修改1次
